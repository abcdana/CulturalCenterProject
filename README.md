## 📝 CulturalCenterProject 프로젝트 후기
<p>프로젝트를 진행하며 나 스스로에게 아쉬웠던 부분은 내가 담당한 기능들의 유효성 검사를 철저히 하지 못했다는 점이다. 처음에 코딩을 시작할 땐 기능 구현에 급급해 Validation을 놓치고 있었는데 뒤늦게 하려니 이곳 저곳에서 오류가 떠 마지막 구동 시에 몹시 당황스러웠다. 또한, 동일한 코드를 두 번 이상 사용하게 될 시엔 메서드를 정의, 호출하여 사용해야 한다는 사실을 알면서도 막상 소스를 짜다 보면 습관이 되어있지 않아서 같은 기능을 두 세번씩 적기도 했었다. 다음 프로젝트부턴 효율적인 개발 습관을 들여야겠다는 생각이 들었다.</p>
<p>마지막으로 팀 구성원들에 나를 포함하여 비전공자들이 많아 초반엔 걱정을 했지만 사전지식이 있던 전공자 조장이 조원들을 잘 이끌어 주었고, 조장을 따라 조원들도 잘 참여해주었다. 그로 인해 이 프로젝트를 기한 내에 무사히 완료하였음에 우리 조 모두에게 감사함을 느낀다.</p>
